# Beeja – Micro-Investment App 🌱

Beeja is a personal finance MVP app that helps Indian users invest spare change automatically. Inspired by Acorns.

## 🚀 Tech Stack
- React Native
- Firebase (Realtime DB + Auth)
- Google Sign-in
- HTML/CSS for waitlist site
